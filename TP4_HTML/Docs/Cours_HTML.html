Introduction au HTML

HTML (HyperText Markup Language) est le langage de base pour créer des pages web. Il permet de structurer le contenu des pages grâce à des balises.

Les balises essentielles

  • <h1> ... </h1> : Définit un titre de niveau 1.
  • <p> ... </p> : Définit un paragraphe.
  • <a href="..."> ... </a> : Définit un lien hypertexte.
  • <img src="..." alt="..."> : Insère une image.

Pour en savoir plus, vous pouvez explorer les différentes balises disponibles et leur usage pour enrichir vos pages web.
