Introduction au CSS

CSS (Cascading Style Sheets) permet de styliser les pages HTML et de contrôler leur mise en page. Avec CSS, vous pouvez changer les couleurs, les polices, la disposition et bien plus encore.

Les sélecteurs importants

  • element { } : Sélectionne les éléments HTML par leur nom (exemple : p { color: red; }).
  • .classe { } : Sélectionne les éléments ayant une classe spécifique (exemple : .important { font-weight: bold; }).
  • #id { } : Sélectionne un élément par son ID unique.

Exemple de mise en forme simple

body {
  background-color: #f0f8ff;
  font-family: 'Arial', sans-serif;
}

Explorez les propriétés CSS pour donner du style à vos pages web.
